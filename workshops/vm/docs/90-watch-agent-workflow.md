# VM Module 6: Watch Agent Workflow

The VM investigation should surface a visible stage-by-stage chain:

1. Observe
2. Investigate
3. Correlate
4. Form hypothesis
5. Propose remediation
6. Await approval
7. Execute (after approval)
8. Validate recovery
9. Generate postmortem

Artifacts written by the tooling:

- `workshops/vm/output/investigation-trace-*.log`
- `workshops/vm/output/postmortem-*.md`
- `workshops/vm/output/actions-audit.log`
