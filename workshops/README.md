# Workshops

This repository now supports two workshop tracks:

- **AKS track** (cloud-native): the original Kubernetes + CosmosDB workshop flow.
- **VM track** (enterprise migration): Windows Server + IIS operational scenarios with approval-gated remediation.

Both tracks follow the same workshop philosophy:

1. Deploy from code.
2. Introduce a realistic fault.
3. Observe investigation and root-cause reasoning.
4. Apply controlled remediation.
5. Capture a postmortem.
